### 资源管理器
此项目是基于C#实现的操作系统-文件管理的大型试验-资源管理器

#### 目的：
实现文件系统资源管理器的设计与开发，完成页面设计并实现其各种功能。
#### 要求：
（1）完成资源管理器的窗体设计以及属性页面设计。
（2）实现窗体内的各种功能：包括获取节点、展开节点获取其子目录并显示、前进后退、返回上一级目录、刷新、搜索、显示对象数、复制、粘贴、剪切、删除、还原等功能。
（3）显示属性页面信息。
